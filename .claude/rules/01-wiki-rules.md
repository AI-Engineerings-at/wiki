# Wiki Rules — Gelten fuer ALLE Agenten

## Bild-Regeln
- **Grosse Kachel/Card:** IMMER echtes Bild (Logo oder generiert). NIE grosses Emoji als Hauptbild.
- **Kleine Inline-Elemente (Tags, Sidebar):** Emoji ist okay.
- **Thumbnails:** 200x150px, WebP Format, max 50KB, mit Alt-Text
- **Hero Images:** Max 500KB, WebP bevorzugt
- **Alle Bilder:** MUESSEN Alt-Text haben (beschreibend, nicht "Bild von...")

## Content-Regeln
- **Quellenangabe:** Jeder Artikel hat Originalquelle + Link + Datum
- **Kein Volltext:** Papers zusammenfassen + eigene Einordnung, NIE kopieren
- **Gesetze:** Gemeinfrei, duerfen zitiert werden — immer mit EUR-Lex/RIS Link
- **Logos:** Open Source = okay fuer Editorial Use. Proprietaer (OpenAI, Google) = Brand Guidelines pruefen
- **Sprache:** Artikel DE + EN, Code EN, Commits EN

## Qualitaets-Regeln (S9-konform)
- Kein Quick-Fix — Root Cause adressieren
- Keine Mock-Daten — echte Inhalte oder gar keine
- Keine Platzhalter — "TODO: implement later" ist verboten
- Kein Auto-Publish — IMMER /approve durch Joe

## DSGVO
- Keine externen Fonts (Google Fonts CDN verboten)
- Keine Tracking-Cookies ohne Consent
- Keine externen Bilder (Hotlinking) — immer lokal speichern
- Alle externen Links: rel="noopener noreferrer"

## SEO
- Jede Seite: <title>, <meta description>, OG Tags, Canonical URL
- Sitemap: ALLE Seiten eingetragen
- Heading-Hierarchie: h1 → h2 → h3 (keine Spruenge)

## Barrierefreiheit (WCAG 2.1 AA)
- Alle <img>: alt Attribut (nicht leer)
- Kontrast: min 4.5:1 Text, 3:1 grosse Elemente
- Keyboard-Navigation: alle interaktiven Elemente erreichbar
- Skip-to-Content Link vorhanden
