# No-Code AI Builders — Comparison

| Builder | Visual | RAG | Agents | Self-Hosted | API |
|---------|--------|-----|--------|-------------|-----|
| **Flowise** | ✅ Nodes | ✅ | ✅ | ✅ | ✅ |
| **Langflow** | ✅ Graph | ✅ | ✅ | ✅ | ✅ |
| **Dify** | ✅ Workflow | ✅ | ✅ | ✅ | ✅ |
| **n8n AI** | ✅ Flow | ⚠️ | ❌ | ✅ | ✅ |

## Flowise

Best for: LLM-specific workflows

Node types: LLM, Memory, Vector Store, Tools

## Langflow

Best for: LangChain-based apps

Direct component integration

## Dify

Best for: Business applications

Team collaboration, publishing

## n8n AI Nodes

Best for: Extend existing n8n workflows

Limited AI but good REST API integration

## Recommendations

- **Pure LLM**: Flowise
- **LangChain-based**: Langflow
- **Business team**: Dify
- **Workflow extension**: n8n AI Nodes
