# No-Code AI Builder — Vergleich

| Builder | Visual Editor | RAG | Agent Mode | Self-Hosted | API |
|---------|---------------|-----|-----------|-------------|-----|
| **Flowise** | ✅ Node-based | ✅ | ✅ | ✅ | ✅ |
| **Langflow** | ✅ Graph | ✅ | ✅ | ✅ | ✅ |
| **Dify** | ✅ Workflow | ✅ | ✅ | ✅ | ✅ |
| **n8n AI Nodes** | ✅ Flow | ⚠️ Limited | ❌ | ✅ | ✅ |

## Flowise

Best für: LLM-spezifische Workflows

Node Types: LLM, Memory, Vector Store, Tools

## Langflow

Best für: LangChain-basierte Apps

Direct LangChain Component Integration

## Dify

Best für: Business-ready Applications

Team Collaboration, Publishing

## n8n AI Nodes

Best für: Bestehende n8n-Workflows erweitern

Limited AI Features, aber gute Integration mit REST APIs

## Empfehlung

- **Reine LLM-Flows**: Flowise
- **LangChain-basiert**: Langflow
- **Business-Team**: Dify
- **Workflow-Erweiterung**: n8n AI Nodes
